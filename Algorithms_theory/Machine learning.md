

## Level 1 — Beginner / Very Easy

These algorithms have simple intuition and minimal math.

### Supervised Learning

- Linear Regression
    
- Logistic Regression
    
- K-Nearest Neighbors (KNN)
    
- Naive Bayes
    
- Decision Trees
    
- Perceptron (single-layer)

- K-Means Clustering
    
- Basic Hierarchical Clustering
    
- PCA (basic conceptual understanding)

- Hidden Markov Models (HMMs)
    

### Manifold Learning

- UMAP
    
- Isomap

- Mixture of Experts
